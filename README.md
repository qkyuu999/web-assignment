# web-assignment Webアサインメント

このリポジトリは、大学院の平和教育プログラミング課題として作成したものです。

## 🔧 使用方法

1. このリポジトリをダウンロードまたはクローンします。
2. `index.html` をブラウザで開くと、ページを確認できます。
